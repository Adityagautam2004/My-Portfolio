# Aditya Gautam - Personal Portfolio 🚀

A modern, responsive, and highly interactive personal portfolio website showcasing professional experience, technical skills, projects, and achievements. Designed with a premium glassmorphism aesthetic and built using React.js and Vite.

## ✨ Features

- **Glassmorphism UI**: Beautiful, translucent, and frosted-glass components for a premium visual and spatial experience.
- **Dynamic Theming**: Seamless light and dark mode toggling.
- **Rich Interactive Animations**: Micro-interactions, hover effects, parallax scrolling, and smooth section transitions that keep the interface feeling responsive and alive.
- **Responsive Layouts**: Carefully crafted adaptive layouts ensuring a seamless experience across desktop, tablet, and mobile devices.
- **Interactive Bento Grid Layout**: Projects and achievements displayed in modern grid formats that mimic dashboard stylings.
- **Easter Eggs 🤫**: Includes hidden interactions like Konami code recognition and interactive multi-click surprises!
- **Clean Component Architecture**: Built using a robust React component structure for easy scalability and maintenance.

## 🛠️ Technology Stack

- **Framework**: [React 18+](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS3 with Custom Variables, CSS Grid, Flexbox, and complex CSS Animations.
- **Icons & Typography**: Clean modern interfaces featuring standard custom variable-based font pairing `Inter` / `Fira Code`.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Adityagautam2004/My-Portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd "My Portfolio"
   ```

3. Install NPM packages:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to the provided local URL (typically `http://localhost:5173`).

## 🎨 Design & Development Principles

- **Aesthetics First**: Every component is polished using precise paddings, glowing effects, and fluid transitions.
- **Accessibility**: Semantic HTML structures and aria labels enable screenreader compatibility.
- **High Performance**: Native CSS usage over heavy JS animation libraries keeps runtime speedy and light.
